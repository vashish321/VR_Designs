On-Page SEO & Technical SEO Setup On-Page SEO is the practice of optimizing pages within a website in order to improve search engine visibility and target keywords for better rankings. This includes optimizing HTML elements, code and content. Technical SEO refers to the process of optimizing your websites infrastructure to improve how search engines crawl and index your website. Common factors which are part of our On-Page SEO & Technical SEO process include:
URL Formatting
The URL is the address of a web page. Often ignored, it’s important to include target keywords in website URL’s and make sure they are search
engine
Title Tags
Title Tags – This HTML element specifies the title of a web page. You can view the title of a web page by hovering over a browsers tab. Title tags are also used to construct your
websites listing in the search results page (SERPs).
Content
– You may have heard the saying “Content is King”. This is probably the most crucial part of SEO that is typically overlooked. High quality, original content that includes
target keywords is important if you’d like to rank for key terms.
Meta Descriptions
Meta descriptions are hidden HTML elements on web pagesthat are set to describe the content on a web page. They are used in creating listings in theSERP’s below a search listings title. We make sure that you’re websites meta
meaningful and contain target key words.
Responsive Design
Google previously launched Mobilegeddon which was an algorithmic update to place priority on responsive websites. We ensure that your website displays correctly on all devices including smartphones, tablets,
laptops and desktops.
